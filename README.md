# Laravel CRUD Student & Classroom

Project CRUD sederhana menggunakan Laravel.

## Fitur
- CRUD Classroom
- CRUD Student
- Relasi One To Many
- Validation
- Foreign Key

## Cara Install

1. Clone repository
2. Jalankan:
   composer install
3. Copy file .env
   cp .env.example .env
4. Generate key:
   php artisan key:generate
5. Atur database di .env
6. Jalankan migration:
   php artisan migrate
7. Jalankan server:
   php artisan serve
